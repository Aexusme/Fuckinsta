Get Instagram account information with this tool

*Things that must be observed !!!*

# Installation

```
git clone https://github.com/Aexusme/Fuckinsta.git
```
Go to the directory
```
cd Fuckinsta
```
See Help
```
python3 Fuckinsta.py --help
```
